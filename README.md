# Shadow-Shuriken-Sprint
Shadow Shuriken Sprint ist ein actiongeladenes 2D-Jump-and-Run, in dem du als lautloser Ninja durch 20 abwechslungsreiche Level sprintest, springst und kämpfst.  Stelle dich gefährlichen Gegnern, zerstöre Blöcke, aktiviere mächtige Effekte durch Münzen und besiege epische Bosse!
